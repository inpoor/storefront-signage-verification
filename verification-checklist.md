# 门头招牌核验检查清单（银行运营 SOP）

## 阶段一：受理阶段

在客户提交招牌照片时，首先确认以下资料完整性：

- [ ] 营业执照副本（清晰可辨）
- [ ] 门头招牌照片（正面拍摄，招牌完整入镜）
- [ ] 经营场所外观全景照片（含门牌号，若有）
- [ ] 照片拍摄时间不早于申请日期前 30 天
- [ ] 以上资料均无明显的文件损坏、无法打开等异常

## 阶段二：机审阶段（自动化）

以下条件触发自动拒绝，不进入人工复核：

1. 图像分辨率低于 640×480 → `自动拒绝 → 要求重新提交`
2. 图像文件包含 EXIF 编辑软件标记（Photoshop、美图秀秀等）→ `标记可疑 → 转人工`
3. 图像哈希与已知造假案例库匹配 → `自动拒绝 → 标记欺诈`
4. 图像哈希与该客户历史提交照片完全一致 → `警告 → 可能重复提交`
5. 招牌文字 OCR 结果为空（无任何可识别文字） → `自动拒绝 → 无企业标识`
6. 招牌文字中检测到"纸"、"板"、"手写"等关键词 → `标记可疑 → 转人工`

## 阶段三：人工复核阶段

人工复核时，按以下顺序逐项检查：

### 3.1 材质确认

| 检查项 | 正常 | 可疑 | 异常 |
|--------|------|------|------|
| 招牌是否有金属/亚克力质感 | ☐ | ☐ | ☐ |
| 是否可见安装螺丝/膨胀螺栓 | ☐ | ☐ | ☐ |
| 边缘是否封装完整 | ☐ | ☐ | ☐ |
| 是否有纸板波纹/泡沫颗粒 | ☐ | ☐ | ☐ |
| 是否有手写笔迹痕迹 | ☐ | ☐ | ☐ |

### 3.2 文字确认

| 检查项 | 结果 |
|--------|------|
| 招牌公司名称 | [填写] |
| 营业执照公司名称 | [填写] |
| 是否一致 | ☐ 完全一致 / ☐ 简繁体一致 / ☐ 不一致 |

### 3.3 环境确认

| 检查项 | 正常 | 异常 |
|--------|------|------|
| 建筑类型是否与经营范围匹配 | ☐ | ☐ |

| 检查项 | 正常 | 可疑 |
|--------|------|------|

## 阶段四：判定与处置

### 处置分级

| 风险等级 | 判定条件 | 处置方式 | 时效要求 |
|---------|---------|---------|---------|
| **一档-正常** | 所有检查项正常 | 自动通过 | 实时 |
| **二档-关注** | 1-2项可疑但可解释 | 标注后通过 | T+1 日 |
| **三档-存疑** | 多项可疑或关键项异常 | 要求客户经理实地核查 | T+3 日 |
| **四档-拒绝** | 确凿伪造证据 | 拒绝并记录欺诈 | 即时 |
| **五档-上报** | 系统性造假特征（AI生成、团伙特征） | 上报反欺诈部门 | 即时 |

### 实地核查触发条件

满足以下任一条件，必须启动实地核查：

1. 招牌材质被判定为简易/临时材料
2. 文字内容与营业执照不一致
3. 检测到 PS/AI 修改痕迹
4. 环境要素与注册地址区域明显不符
5. EXIF 元数据完全缺失 + 画面存在 2 项以上可疑特征
6. 该客户有历史造假记录

### 实地核查要点

客户经理实地核查时需拍摄：

- [ ] 含 GPS 定位的门头全景照片
- [ ] 招牌材质特写（近距离拍摄以看清表面纹理）
- [ ] 招牌安装固定方式特写
- [ ] 门牌号与招牌同框照片
- [ ] 经营场所内景照片（含办公/生产/仓储区域）
- [ ] 客户在经营场所内的合影（含当日日期标识）

## 附：常见违规判定速查

| 违规类型 | 核心判定依据 | 处置 |
|---------|-------------|------|
| 纸板招牌 | 可见瓦楞结构/纸张折痕/手写文字 | 四档-拒绝 |
| PS换字 | 光照不一致/压缩块异常/边缘过度平滑 | 四档-拒绝 |
| AI生成 | 文字乱码/结构错误/光影矛盾/细节平滑 | 五档-上报 |
| 旧照翻拍 | 摩尔纹/屏幕反光/显示器边框 | 三档-存疑 |
| 异地冒用 | 建筑风格不符/车牌不符/GPS不符 | 四档-拒绝 |
| 重复提交 | 图像哈希匹配历史记录 | 三档-存疑 |
