# 门头招牌真实性核验（银行版）

## 概述

银行在开户、授信、贷后管理等环节中，要求客户提交经营场所"门头招牌"照片作为经营真实性的佐证。本 Skill 提供一套结构化的多模态核验流程，通过「视觉特征分析 + 元数据检测 + 逻辑一致性校验」三重手段识别伪造的招牌照片。

## 核验触发场景

以下任一情况出现时，应激活本 Skill 进行门头招牌核验：

- 银行客户提交门头招牌照片作为经营证明材料
- 对公账户开户前的实地经营核验
- 授信审批中的经营场所真实性验证
- 贷后管理中定期回传的经营状态核查
- 反欺诈系统中触发的招牌照片复查

## 核验工作流

严格按以下步骤执行，每步输出判定结论和置信度：

### 第一步：图像基础质量评估

先对照片进行基础质量筛查，排除明显不合格的图像：

1. **分辨率检查**：分辨率低于 640×480 或文件大小小于 20KB，标记为 `低质量图片`
2. **模糊度检测**：画面严重模糊、无法辨认文字内容，标记为 `模糊不可辨`
3. **拍摄角度检查**：极端仰角/俯角导致招牌变形严重，标记为 `拍摄角度异常`
4. **遮挡检查**：招牌区域被物体、人物、反光大面积遮挡，标记为 `遮挡`

若基础质量不通过，直接返回 `核验不通过 → 需重新拍摄提交`，不进入后续步骤。

### 第二步：招牌材质与工艺分析（核心）

此步骤是本 Skill 的核心，重点识别「低成本伪造」行为：

#### 2.1 纸板/简易材质检测

检查招牌是否属于下列伪造特征：

| 伪造类型 | 视觉特征 | 判定权重 |
|---------|---------|---------|
| **瓦楞纸板** | 边缘可见波浪形夹层结构、表面粗糙、有纸板折痕 | 极高 |
| **普通纸板/卡纸** | 表面反光均匀、边缘可见纸张厚度、易弯折痕迹 | 极高 |
| **泡沫板（KT板）** | 表面哑光、边缘有泡沫颗粒、轻质感 | 高 |
| **喷绘布（随意悬挂）** | 边缘不规则裁剪、无边框固定、褶皱明显 | 高 |
| **横幅/条幅** | 布料纹理、边缘有线头、字体印刷粗糙 | 高 |
| **A4纸拼接** | 多张纸拼接痕迹、接缝明显、颜色不一致 | 极高 |
| **手写文字** | 笔迹不规整、墨色不均匀、有涂抹修改痕迹 | 极高 |

#### 2.2 正式招牌特征对照

正规门头招牌应具备以下特征，逐项核对：

- [ ] 材质：亚克力、铝塑板、不锈钢、发光字、PVC等耐用材料
- [ ] 边框/包边：有金属或塑料边框封装，非裸露边缘
- [ ] 安装方式：膨胀螺丝固定在墙面上，有安装痕迹
- [ ] 字体工艺：印刷/雕刻字体工整，笔画均匀，无毛边
- [ ] 灯光：夜间模式应有背光/发光效果（若照片为夜间拍摄）
- [ ] 完整性：招牌四边完整，未被画面裁切

#### 2.3 招牌与环境一致性

- 招牌尺寸与门面比例是否协调（纸板招牌通常尺寸偏小）
- 招牌安装位置是否在门头合理区域（门楣上方或侧方）
- 招牌风格是否与周边建筑环境一致
- 招牌新旧程度是否与建筑外观匹配

### 第三步：文字内容核验

#### 3.1 招牌文字提取

- 提取招牌上的所有文字内容（公司名称、简称、经营范围描述等）
- 识别文字的语言、字体、排列方式

#### 3.2 工商信息比对

将提取的文字与客户提供的以下信息逐一比对：

- 营业执照上的企业名称
- 统一社会信用代码
- 注册地址
- 法定代表人姓名（若招牌上有显示）

比对规则：
- 企业名称完全一致 → `通过`
- 企业名称存在简繁体差异但实质相同 → `通过（简繁体差异）`
- 企业名称存在字词增减 → `不通过（名称不一致）`
- 招牌无任何公司名称 → `不通过（无企业标识）`

### 第四步：图像篡改检测

检测照片是否存在伪造/修改痕迹：

#### 4.1 常见篡改手法

| 篡改手法 | 检测方法 | 视觉线索 |
|---------|---------|---------|
| **PS抠图换字** | ELA（误差水平分析）、噪声一致性分析 | 文字区域边缘模糊、光照方向不一致、分辨率突变 |
| **AI生成图片** | 检测生成对抗网络伪影 | 细节过度平滑、"不自然感"、背景元素逻辑错误 |
| **异地招牌冒用** | EXIF GPS 比对、场景元素分析 | 建筑风格不匹配、植被/气候不符、车牌号地域不对 |
| **旧照翻拍** | 屏幕摩尔纹检测、二次压缩特征 | 可见显示器像素格、屏幕反光、边框边缘 |
| **重复使用同一照片** | 图像哈希比对该客户历史提交 | 与历史照片完全一致或高度相似 |

#### 4.2 EXIF / 元数据检查

检查照片文件的元数据：

- 拍摄时间是否合理（不应早于开户申请日期太久）
- 拍摄设备信息是否存在（缺失元数据为可疑信号）
- GPS 坐标是否与注册地址区域一致（若有）
- 是否有 Photoshop/美图秀秀等编辑软件的痕迹
- 文件修改时间与拍摄时间是否吻合

### 第五步：综合判定与报告生成

综合以上四步的检测结果，按以下规则生成最终判定：

#### 判定规则

| 条件 | 判定结果 | 处置建议 |
|------|---------|---------|
| 纸板/手写类造假特征命中 1 项以上 | `高风险 - 疑似伪造` | 拒绝，要求客户经理实地核查 |
| PS/AI生成痕迹命中 1 项以上 | `高风险 - 疑似伪造` | 拒绝，转反欺诈调查 |
| 企业名称比对不通过 | `高风险 - 信息不符` | 拒绝，核对营业执照 |
| 多处可疑特征但无确凿证据 | `中风险 - 待人工复核` | 转人工坐席复核 |
| 元数据异常但视觉正常 | `中风险 - 待人工复核` | 要求重新拍摄并开启GPS |
| 所有检测通过 | `低风险 - 核验通过` | 正常进入下一流程 |

## 使用方式

当用户上传门头招牌图片并请求核验时：

1. 先加载 `references/fake-signage-patterns.md` 了解常见伪造模式
2. 按上述五步工作流逐步分析
3. 每步给出明确结论和置信度
4. 最终输出结构化核验报告
5. 对中/高风险案例，给出具体的处置建议

## 注意事项

- 对高风险的判定需给出具体的视觉证据描述，不能仅凭"感觉"
- 若图片质量不足以做判断，应明确要求客户重新提交，而非强行判定
- 银行合规要求下，核验报告需留存完整痕迹，不得截断或省略关键信息
- AI生成图片检测存在误判可能，高风险判定需结合材质/文字多维度确认
